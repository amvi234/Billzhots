# Billzhots APP

## Description:
A NextJs and Django Web App supporting MFA and AI integration. Get AI prompted responses on the product research, inspection, analysis, reviews and filtering.

## TECH used

- NEXTJS
- DJANGO
- TYPESCRIPT
- PYTHON
- TAILWINDCSS
- DOCKER

## FEATURES

- Multi Factor Authentication
- AI Integration
