# Billzhots APP

## Description:
A NextJs and Django Web App supporting MFA and AI integration. Get AI integrated bills amount calculation with scanned images.

## TECH used


- NEXTJS
- DJANGO
- TYPESCRIPT
- PYTHON
- TAILWINDCSS
- DOCKER
